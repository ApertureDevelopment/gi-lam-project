Die Idee hinter dem Sortieralgorithmus ist simpel. In einer for-Schleife werden alle, bis auf das letzte Element durchlaufen. Dabei wird das ausgewählte Element mit dem nachstehenden Element verglichen. Wenn das zweite Element größer ist als das Ausgewählte, wird ein Tausch durchgeführt. Da ein Durchlauf nicht ausreicht und die Menge an benötigten Durchläufen theoretisch variieren könnte, wurde um die for-Schleife eine while-Schleife konzipiert und ein boolean eingebaut, der die Schleife beendet. Am Anfang jeder While-Schleife wird der Wert auf False gesetzt, in der If-Bedingung auf True, sobald eine Änderung vorgenommen werden muss. Wenn die While-Schleife das erste Mal durchläuft, ohne dass eine Sortierung vorgenommen werden muss, bleibt der Wert damit auf False und die Funktion findet ihr Ende.
